SaaS Model Identity and Authentication Platform

IDXaaS (Identity as a Service) is a comprehensive identity and authentication platform designed for administrative, banking, and corporate clients. The platform offers a modular system with standard communication interfaces for user authentication and data management solutions, enabling organizations to implement robust identity management without building infrastructure from scratch.

This major EU-funded project was completed on November 29, 2020, delivering a production-ready identity platform prepared for Software-as-a-Service operation across multiple sectors.

EU Funding and Development

The project received 67,626,000 HUF in total funding, with 64.08% support from the European Union through the VEKOP program (Project ID: VEKOP-2.1.7-15-2016-00606). This investment enabled E-Group to develop enterprise-grade identity infrastructure that meets the demanding requirements of government, banking, and corporate sectors.

The development was conducted in collaboration with the University of Pécs, combining academic research expertise with E-Group's practical experience in building national-scale identity systems.

Modular Architecture and Integration

IDXaaS implements a modular design with standard communication interfaces, enabling seamless integration with existing enterprise systems:

  • Standard API interfaces for system integration
  • Modular components for flexible deployment
  • Context-based authentication capabilities
  • Scalable SaaS architecture
  • Multi-tenant support for diverse client needs

Context-Based Authentication

The platform features advanced context-based authentication capabilities that adapt security requirements based on user behavior, location, device, and transaction risk. This intelligent approach balances security with user experience, applying appropriate authentication levels for different scenarios.

This context-aware system reduces friction for routine operations while maintaining high security for sensitive transactions, making it ideal for banking and administrative applications.

Target Sectors and Applications

IDXaaS is specifically designed for three key sectors:

  • Administrative sector — Government and public administration identity management
  • Banking sector — Financial institution authentication and compliance
  • Corporate sector — Enterprise identity and access management

Each sector benefits from the platform's modular architecture, allowing customization while maintaining core security and compliance standards.

SaaS Model and Scalability

Built as a true Software-as-a-Service platform, IDXaaS eliminates the need for organizations to maintain their own identity infrastructure. The platform provides:

  • Cloud-native architecture for elastic scalability
  • Multi-tenant isolation and security
  • Subscription-based deployment models
  • Continuous updates and security patches
  • Compliance with European data protection regulations

Academic Partnership and Innovation

The collaboration with the University of Pécs brought academic rigor to the platform's development, ensuring that IDXaaS incorporates the latest research in identity management, cryptography, and authentication technologies. This partnership represents E-Group's commitment to bridging academic research with practical, production-ready solutions.

The Broader Digital Identity Platform

IDXaaS is the SaaS delivery layer of E-Group's Digital Identity Platform — a comprehensive product family with multiple pillars:

  • IDX Group — identity service integration (e-ID card, biometric data), data source integration (central data sources), and ID federation services for Single Sign-On
  • SDX — digital signature, advanced e-Signature creation and verification, including a post-quantum security algorithm already in production
  • IOX / SPG — embedded workflow engine enabling flexible orchestration of business logic
  • Electronic registered delivery for e-documents

The platform is built as a 12-factor cloud-native solution, compliant with eIDAS and GSMA directives. MyD — the white-labeled mobile application — is the user-facing front-end of this entire platform.

Technical Capabilities

The Digital Identity Platform includes capabilities far beyond standard authentication:

  • NFC-based e-ID card reading with issuer verification
  • AI-enriched OCR for ID card data capture
  • Face comparison — embossed ID photo versus live image
  • Biometric verification with face recognition and liveness detection (powered by E-Group AI)
  • Live-video analysis with TensorFlow model
  • ML-driven age detection and object classification
  • Signature reading from ID cards and placement on PDF documents
  • Speech-to-Text supported input
  • Root detection, jailbreak warning, device eligibility checks
  • Geolocation detection, accelerometer and motion sensing

Real Deployments at National Scale

Diákhitel (Hungarian Student Loan Center): 95% of all Hungarian student loans are contracted through the Digital Identity Platform. National-scale identity verification, digital contracting, and KYC compliance for the entire Hungarian higher education financing system.

Hungarian National Bank (MNB): Secure authenticated data transfer for Hungary's central bank. Registration forms, electronic signature certificates, access to MNB notifications and publications, mandatory reporting, and integration with AVDH central government notary services.

e-Voting for regional government: Pre-election registration, petition collection, and electronic voting system. Developed with the Hungarian Association for Electronic Signature. Biometric verification, e-ID reading, legally binding digital signatures, and scalable architecture for high-volume election traffic.
